INTRODUCTION

ELIXIR is an AI which quizzes the user inorder to test the understanding of the code. The user can manually provide the input code for debugging. It quizzes the user regarding the input code and provides the debugged version only if the user manages to answer minimum levels of questions appropriately.

TERMS AND CONDITIONS:

Here is how elixir works differently

  1. The quiz involves multiple choice questions based on the code provided by the user.
  2. Elixir will generate questions based on the number of lines of code.
  3. The user will have access to the debugged code only if he/she manages to answer minimum questions correctly
  4. User can skip to the next question if not attempted to answer the previous question
  5. This tool aims to reduce mindless copying of code by quizzing the user on the same.
